# Delete Temporary Files

This script allows you to remove temporary files from your system.

## Steps

Step 1: 

    ```bash
    git clone https://github.com/saifsahriar/delete-temp-files.git
    ```

Step 2:

    ```bash
    cd delete-temp-files
    ```

Step 3:

    ```bash
    chmod +x remove.sh
    ```

Step 4:

    ```bash
    ./remove.sh
    ```
